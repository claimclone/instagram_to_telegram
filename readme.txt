Данный бот служит для извлечения (скрейпинга) контента из аккаунтов Instagram
в канал, группу или личную переписку Telegram.

Поддерживается извлечение любых аудио/видео/описательного текста со страниц
заданных профилей пользователей, в том числе приватных.

Для настройки бота необходимо указать все требуемые параметры в файле
конфигураций config.ini. Понадобятся API Token бота, а также действительные
логин/пароль владельца аккаунта Instagram, от имени которого будет
осуществляться процесс скрейпинга.

Если бот должен пересылать сообщения в канал или группу Telegram, то его
следует назначить администратором сообщества.

Для отладки и настройки предусмотрены следующие параметры командной строки:
    --test      : тестовый запуск скрипта - происходит однократный скрейпинг
                  по одной записи с каждой страницы Instagram с пересылкой
                  в Telegram;

    --setup     : осуществляет однократный скрейпинг по одной записи с каждой
                  страницы Instagram, без пересылки в Telegram; используется
                  для настройки пересылки только нового контента в будущем;

    --singlerun : однократный запуск скрипта, без инициации бесконечного цикла.

Примеры использования команд:
    python bot.py --test
    python bot.py --setup

Есть два способа запустить бота в режиме циклической проверки обновлений. Более
простой подразумевает запуск в режиме бесконечного цикла (фоновый процесс). Для
этого нужно вызвать скрипт без параметров:
    python bot.py

Другой способ — это периодический вызов исполняемого скрипта через планировщик
(crontab). Для этого служит команда:
    python bot.py --singlerun

Все необходимые зависимости для установки перечислены в файле requirements.txt.
